# FashionStoreHub

## Project Overview
FashionStoreHub is a Java Full Stack E-Commerce web application developed using Java, JSP, Servlets, JDBC, and MySQL. The project allows customers to browse products and enables administrators to manage products and categories.

## Technologies Used
- Java
- JSP & Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat
- Eclipse IDE
- Git & GitHub

## How to Run
1. Clone this repository.
2. Import the project into Eclipse.
3. Configure Apache Tomcat.
4. Create the MySQL database.
5. Import the SQL script.
6. Update the database credentials in `DBConnection.java`.
7. Run the project on Tomcat.
